# Trash

## View Trash
- Click the trash icon in the sidebar footer
- The trash view should show deleted notes with title and deletion timestamp
- If empty, show "Trash is empty" message

## Restore Note
- Hover a trashed note, click the restore icon (RotateCcw)
- The note should disappear from trash and reappear in the sidebar

## Permanent Delete
- Hover a trashed note, click the X icon
- A confirmation dialog should appear
- Confirm — the note should be permanently removed

## Mobile Trash
- Set viewport to a phone size such as 390x844
- Trash view should fit without horizontal scrolling
- Restore and permanent delete actions should be visible/tappable without hover
