# Focus Mode

## Enter Focus Mode
- Click the maximize icon in the mode switcher (below the separator)
- All chrome should fade away: sidebar, mode switcher, word count, sidebar expand button
- An "Esc to exit focus mode" hint should appear briefly at the top center

## Exit Focus Mode
- Press Escape — all chrome should fade back in
- OR move mouse to the top of the screen — the exit hint reappears, click it to exit

## Editor Still Works
- While in focus mode, typing should work normally
- Auto-save should still function
