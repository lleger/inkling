# Version History

## View Versions
- Open a note that has been edited over time
- Open Cmd+K, select "Version history"
- A version list should appear on the left with timestamps and previews
- "No versions yet" if the note hasn't had versions created (requires 5+ min between edits)

## Preview a Version
- Click a version in the list
- A read-only rich text preview should appear on the right
- A "Restore this version" button should be visible

## Restore
- Click "Restore this version"
- The note should update with the old content
- A toast "Version restored" should appear
- The editor should reopen with the restored content

## Back Navigation
- Click the back arrow — should return to the note editor
