# Command Palette

## Open/Close
- Press Cmd+K — palette should appear with search input focused
- Press Escape — palette should close
- Click the backdrop — palette should close

## Actions
- With no query, palette shows actions (New note, Go home, mode switches, etc.) and recent notes
- Type "new" — "New note" action should appear
- Type "settings" — "Settings" action should appear
- Type "trash" — "Trash" action should appear

## Note Search
- Type a note title — matching notes should appear under "Notes" section
- Click a note — palette closes, note opens in editor

## Create with Title
- Type a name that doesn't match any action or note
- A "Create <name>" option should appear at the bottom
- Select it — a new note should be created with that title as a heading

## Keyboard Navigation
- Press ArrowDown/ArrowUp to move selection highlight
- Press Enter to execute the selected action
